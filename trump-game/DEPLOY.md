# 升级游戏 - 部署到 Render（免费 24h 在线）

> 前后端已打包在一起，部署后得到一个链接，往微信里一丢就能玩。

## 第 1 步：把代码推送到 GitHub

1. 去 https://github.com/new 创建一个公开仓库（比如叫 `trump-game`）
2. 把 `trump-game/` 这个文件夹里的所有文件传上去（拖拽上传或 git push）

## 第 2 步：在 Render 上点一下部署

1. 打开 https://render.com （用邮箱注册，免费）
2. 点击右上角 **New +** → **Blueprint**
3. 连接你刚才创建的 GitHub 仓库
4. Render 会自动识别 `render.yaml`，直接点 **Apply**
5. 等 2 分钟，状态变 **Live** 后，复制链接

## 第 3 步：开玩

- 把链接发到微信，好友点开即玩
- 约定同一个房间号（如 8888）进入同桌
- 4 人齐了就点准备 → 房主点开始

---

**注意**：Render 免费版 15 分钟没人访问会休眠，下次打开要等 20~30 秒唤醒。如果你们要连续玩，不受影响。
